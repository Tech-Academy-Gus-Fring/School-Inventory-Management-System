# Frontend Startup for School Inventory Management (React)

This workspace now includes a React frontend (Vite-based) for the existing backend.

## Run
1. Start backend as described in root README (`npm start` in backend).
2. In the frontend folder:
   - `cd frontend`
   - `npm install`
   - `npm run dev`
3. Open the URL shown by Vite (default: `http://localhost:3000`).

## Features
- Login / Register
- View equipment list
- Submit equipment request
- Profile display
- Admin-only create equipment

## Notes
- Backend base URL is `http://localhost:5000` in `src/App.jsx`.
- Change if backend uses different host/port.
